# Stack

- **Framework**: Next.js 15.4.9 (App Router)
- **Language**: TypeScript 5.9.3
- **Styling**: Tailwind CSS 4.1.11, tailwind-merge, clsx
- **Animation**: motion, tw-animate-css
- **Database/Backend**: Convex 1.35.1
- **Authentication**: Better Auth 1.5.3 (via @convex-dev/better-auth)
- **AI**: @google/genai 1.17.0
- **Forms**: react-hook-form, @hookform/resolvers
- **Icons**: lucide-react
- **UI Architecture**: React 19.2.1, class-variance-authority
