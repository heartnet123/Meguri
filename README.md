# Meguri / SmartStock

The complete operational backbone for recipe-based businesses and smart retail.

Meguri helps you manage stock, recipes, and day-to-day operations without the usual chaos.

## Features

- Inventory tracking
- Recipe-based stock management
- Smart retail operations support
- Clean workflow for running a business from one place

## Getting Started

### Prerequisites

- Node.js
- npm
- A Gemini API key

### Install

```bash
npm install
```

### Configure environment

Copy `.env.example` to `.env.local` and set your values:

```bash
GEMINI_API_KEY=your_gemini_api_key
```

### Run locally

```bash
npm run dev
```

Then open the app in your browser.

## Build

```bash
npm run build
```
