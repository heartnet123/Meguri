
## 2024-05-31 - Search Input Enhancement
**Learning:** Adding a subtle search icon, clear `aria-label`, and defined focus states drastically improves the visual affordance and accessibility of standalone search inputs.
**Action:** Always wrap search inputs in a relative container to position an icon, apply `type="search"`, and use consistent focus ring styles (`focus:ring-2 focus:ring-accent/10 focus:border-accent transition-all`) to guide users visually and assist screen readers.
